Analysis can be performed from within SpikeDB by writing Python (version 2) analysis scripts and extecuting them from within the Analysis tab.

The following variables are available to you in your analysis script.


Name: Cells
Type: Dictionary
Keys: AnimalID: string
	  CellID: number
      CarFreq: number (in Hz)
	  Threshold: number (in dB SPL)
	  Depth: number (in um)
